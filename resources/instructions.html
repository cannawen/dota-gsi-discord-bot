What is this?

Dota coach discord bot using Valve's Gamestate integration API that reminds you of important timings in the game (roshan timer, wisdom rune spawning, etc.)

How to set up Dota Coach

(You only have to do this once)
  1. Add bot to your server (you must be an admin)
  2. Go to any channel and type /config
  3. Open a text editor and save the config file as gamestate_integration_dota2-coach.cfg on your desktop
  4. Open your Steam library
  5. Right click Dota 2, click "Properties"
  6. Make sure you are on "General" on the left sidebar
  7. Add -gamestateintegration to the launch options
  8. Click on "Local Files" on the left sidebar
  9. Click "Browse..."
  10. Open game/dota/cfg
  11. If it does not already exist, create a new folder called gamestate_integration
  12. Drag and drop the file from step 3 into this new folder
  13. Restart Steam

How to start Dota Coach

(You must do this every time)
  1. Go to a voice channel chat and type /coachme
  2. Join the voice channel to recieve general tips (i.e. rosh respawn time) and click the link provided to recieve private tips (i.e. reminder to use your Trusty Shovel)
  3. When you are done, you should type /stop to have the bot leave the channel

  • If you type /coachme on a non-voice channel, you will only recieve a private coaching link with no discord voice integration. Once you have your link, you can bookmark it and start private coaching directly from the web page

    • Voice commands

    The bot will respond to the following voice commands in a game:
    • "What is roshan timer/status?"
    • "Torment has fallen" - must notify bot by voice when you kill a tormenter (the bot does not automatically get notified)
    • "What is torment timer/status" or "Torment status" - only accurate if the above command has happened
    • "What should I buy" - suggests a random item
    • "What should I buy for # gold" - suggests a random item around # gold (patch 7.32e)
    The speech recognition is terrible at recognizing tormenter so just call it a torment

    Notes

    • This bot is still in beta. A new version of the bot is deployed daily at 7am UTC, but occasional hotfixes may be pushed throughout the day. If you are playing a game at this time, the bot will restart and it may lead to unexpected behaviour
    • If two people use the /coachme command in the same voice channel, only one person will be allowed to play audio through discord.
    • Turbo mode, custom games, or spectating a game will lead to undefined behaviour.
